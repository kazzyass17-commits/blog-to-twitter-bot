# X Developer Portal 公式ドキュメントとFAQ

## 403エラーや認証問題を解決するために読むべきドキュメント

### 1. 認証に関するドキュメント

#### About authentication
- **URL**: Twitter Developer Portalの「Helpful docs」セクションから「About authentication」をクリック
- **内容**: 認証の基本、OAuth 1.0a、OAuth 2.0、Bearer Tokenの使い方
- **重要度**: ⭐⭐⭐⭐⭐

#### Authentication best practices
- **URL**: Twitter Developer Portalの「Helpful docs」セクションから「Authentication best practices」をクリック
- **内容**: 認証のベストプラクティス、セキュリティの推奨事項
- **重要度**: ⭐⭐⭐⭐

### 2. アプリの権限に関するドキュメント

#### App permissions
- **URL**: Twitter Developer Portalの「Helpful docs」セクションから「App permissions」をクリック
- **内容**: アプリの権限設定、Read/Write権限の説明
- **重要度**: ⭐⭐⭐⭐⭐
- **403エラーの原因を理解するために重要**

### 3. 認証情報に関するドキュメント

- **重要度**: ⭐⭐⭐

#### Bearer Tokens
- **URL**: Twitter Developer Portalの「Helpful docs」セクションから「Bearer Tokens」をクリック
- **内容**: Bearer Tokenの取得方法、使用方法
- **重要度**: ⭐⭐⭐⭐

#### Access Token and Secret
- **URL**: Twitter Developer Portalの「Helpful docs」セクションから「Access Token and Secret」をクリック
- **内容**: Access Tokenの取得方法、再生成のタイミング
- **重要度**: ⭐⭐⭐⭐⭐
- **403エラーの原因を理解するために重要**

### 4. プロジェクトとアプリに関するドキュメント

## 403エラーを解決するために特に重要なドキュメント

### 優先度1: 必ず読むべき

1. **App permissions** - アプリの権限設定について
2. **Access Token and Secret** - Access Tokenの再生成について
3. **Error Codes** - 403エラーの詳細について

### 優先度2: 理解を深めるために

4. **About authentication** - 認証の基本
5. **Authentication best practices** - 認証のベストプラクティス

### 優先度3: 参考情報として

6. **About Apps** - アプリの状態管理
7. **Rate Limits** - レート制限について

## アクセス方法

### Twitter Developer Portalから

1. Twitter Developer Portalにログイン
2. アプリを開く
3. 「Keys and tokens」タブを開く
4. 右側の「Helpful docs」セクションから各ドキュメントにアクセス

### 直接URLで

- X API Documentation: https://developer.twitter.com/en/docs
- Error Codes: https://developer.twitter.com/en/docs/twitter-api/error-codes
- Rate Limits: https://developer.twitter.com/en/docs/twitter-api/rate-limits

## 403エラーに関する具体的な情報

403エラーについて調べる場合、以下のキーワードで検索してください：

- "403 Forbidden"
- "App permissions"
- "Access Token regeneration"
- "OAuth 1.0a authentication"
- "API v2 authentication"
